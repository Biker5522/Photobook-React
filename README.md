# Photobook-React
A social network site. User can log in, add new posts, comment them, and manage images albums.
# Build With
* React.js
* Next.js
* Tailwind.css
* react-icons
# Use it yourself
1. Clone the repository
2. Change the working directory <br>
`cd frontend`
3. Install dependencies<br>
`npm i`
4. Run app<br>
`npm run dev`
